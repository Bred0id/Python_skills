### Задача 1. Декоратор для трассировки возвращаемого значения
- **Условие:** Написать декоратор `trace`, который выводит сообщение:  
`Функция <имя_функции> вернула значение <значение>`
после вызова функции.

### Задача 2. Декоратор с повторными попытками выполнения (retry)
- **Условие:** Написать параметризованный декоратор `retry(attempts, delay)`, который при возникновении исключения повторяет вызов функции. При неудачной попытке выводится сообщение:
`Попытка <номер_попытки> завершилась неудачей: <сообщение_исключения>. Повтор через <delay> сек...`
- Если функция не выполнится успешно после заданного числа попыток – выбрасывается последнее исключение.


### Задача 3. Регистрация команд с использованием декоратора
- **Условие:** Написать декоратор `register_command`, регистрирующий функции в глобальном реестре команд, и функцию `run_command(name)` для вызова зарегистрированных функций.
- Он должен хранить названия команд в глобальной переменной `command_registry` - неком словаре.
